# Calendar App with React, 8Base and Azure Functions

## Sections
1. Client app with React `./`
2. 8Base trigger `./8base`
3. Azure Functions Scheduler `./serverless`

## Getting started

1. Clone
2. Run `npm install`
3. Run `npm start` and the app should be at port 3000
